Proyecto: Creación de Tarjetas de Películas

Este proyecto se centra en la creación dinámica de tarjetas de películas utilizando tecnologías web como HTML, CSS, JavaScript y Node.js. A través de este proyecto, hemos aprendido a:

  • Crear una interfaz de usuario interactiva utilizando HTML y CSS.
  • Utilizar JavaScript para manipular el DOM y agregar funcionalidades dinámicas.
  • Conectar nuestra aplicación a una base de datos MongoDB para almacenar y recuperar datos de películas.
  • Utilizar Bootstrap para aplicar estilos predefinidos y mejorar la apariencia de nuestra página.

Además de aprender sobre tecnologías web y bases de datos, este proyecto nos ha permitido desarrollar habilidades en diseño de interfaces, trabajo en equipo y resolución de problemas.
